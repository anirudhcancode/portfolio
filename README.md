## Projects

### 01 — Fraud Detection Pipeline & ML System
End-to-end fraud detection system processing 284,807 real credit card transactions using PySpark for distributed ETL, XGBoost and Random Forest with class weighting, deployed via FastAPI on Docker.
- **ROC-AUC:** 0.98 | **Fraud Recall:** 83% | **Response time:** <200ms
- **Live API:** fraud-detection-pipeline-p6mq.onrender.com/docs
- **GitHub:** github.com/anirudhcancode/fraud-detection-pipeline

### 02 — LLM Financial Intelligence Platform
NLP pipeline using BART, FinBERT, and YAKE to automatically analyze earnings call transcripts — extended into a full financial intelligence platform with 10 years of historical data, ML predictions, and 14 interactive visualizations across 10 companies.
- **94.88%** top confidence | **28,877** price records | **1,849** market events detected
- **Live API:** llm-financial-analyzer.onrender.com/docs
- **GitHub:** github.com/anirudhcancode/llm-financial-analyzer

### 03 — A/B Testing & Experimentation Pipeline
End-to-end experimentation platform simulating a 14-day e-commerce A/B test across 10,000 users, applying z-test for statistical significance, and exposing results via REST API.
- **P-value:** 0.000007 | **Conversion lift:** +26.3% | **Revenue lift:** +$8,528
- **Live API:** ab-testing-pipeline.onrender.com/docs
- **GitHub:** github.com/anirudhcancode/ab-testing-pipeline

### 04 — AI Companion Application *(In Development — Private)*
Cross-platform AI assistant with persistent semantic memory (vector embeddings + pgvector), three-layer emotion detection pipeline, and 10+ integrations including Whoop, Spotify, Gmail, Calendar, and navigation — live on iOS, Android, and web.
- **Demo Page:** anirudhcancode.github.io/portfolio/ai-companion-demo.html
